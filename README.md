# Stubble_Burning
A high-precision burn scar detection system leveraging Sentinel-2 imagery and Google Earth Engine. Features a definitive "Triple-Check" algorithm (dNBR + BAI + dNDVI) with agricultural masking, USGS severity thresholds, and a FastAPI dashboard for dynamic ROI analysis.
